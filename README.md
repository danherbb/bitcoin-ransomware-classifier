# Bitcoin Ransomware Classifier

## Overview
This project analyzes blockchain transaction behavior to classify Bitcoin addresses as ransomware-related or legitimate.

## Dataset
- BitcoinHeist Ransomware Dataset (Kaggle: https://www.kaggle.com/datasets/sapere0/bitcoinheist-ransomware-dataset)

## Workflow
1. Data exploration and preprocessing
2. Feature scaling and preparation
3. Model training and evaluation

## Setup
1. Download dataset from Kaggle
2. Place in 'data/'
3. Run preprocessing notebook
